DATABASE — 6 TABLES CREATED
─────────────────────────────
sites         — id, domain, keyword_1/2/3, category, status,
                articles_count, created_at, updated_at
articles      — id, site_id (FK), title, description, body,
                keyword, category, originality_score, status,
                google_indexed, bing_indexed, created_at, updated_at
media         — id, filename, filepath, type, size, tags, created_at
article_media — id, article_id (FK), media_id (FK) [pivot table]
logs          — id, site_id, article_id, event, message,
                status, created_at
settings      — id, key, value, updated_at

CONTROLLERS — 6 CREATED
────────────────────────
SchedulerController.php
  - run() — loops active sites, reads keywords,
    queues article generation per keyword per day

ArticleGeneratorController.php
  - generate() — builds prompt, calls Groq API skeleton,
    returns structured article data
  - buildPrompt() — constructs SEO article prompt with
    keyword and category

OriginalityController.php
  - check() — runs originality check against threshold,
    returns pass/fail with score and action

PublisherController.php
  - publish() — saves article to DB, attaches media by
    keyword tag, increments site article count, logs event
  - attachMedia() — finds media by keyword tag, links to article

IndexerController.php
  - index() — submits URL to Google and Bing indexing
    API skeletons, updates indexed flags in DB

TelegramController.php
  - send() — checks event config, formats message with
    emoji, calls Telegram Bot API skeleton, logs to DB
  - formatMessage() — formats log with emoji and timestamp

MODELS — 5 CREATED
───────────────────
Site.php
  - Relationships: hasMany Articles
  - Accessors: getKeywordsAttr (returns keywords as array)
  - Scopes: active()

Article.php
  - Relationships: belongsTo Site, belongsToMany Media
  - Scopes: published(), failed(), pending()

Media.php
  - Relationships: belongsToMany Articles
  - Accessors: getTagsArrayAttr (returns tags as array)
  - Scopes: images(), videos()
  - Static: findByKeyword()

Log.php
  - Relationships: belongsTo Site, belongsTo Article
  - Scopes: errors(), today()
  - Static: record() helper for quick log creation

Setting.php
  - Static: get(), set(), setMany()
  - No relationships — pure key/value config store
